# refactor.md

Use this prompt for targeted refactoring only.

## Refactor goals
- reduce duplication
- improve clarity
- improve naming consistency
- improve maintainability
- preserve behavior

## Hard constraints
- no scope expansion
- no new major dependencies
- no contract changes unless explicitly requested
- no UX flow changes unless explicitly requested
- preserve validation and trace logic

## Output format
- Refactor goal
- Candidate files
- Proposed changes
- Risk level
- Implementation
